# 컴포넌트
: 리액트의 재사용 가능한 UI 요소

## 컴포넌트 정의 방법
1. 클래스 컴포넌트
    ```
        class Product extends React/component{
            // 컴포넌트 정의...
        }
    ```
2. 함수형 컴포넌트
    ```
        const Product = () => {
            // 컴포넌트 정의...
        }
    ```

## 컴포넌트 주요 구성 요소
1. state
2. props

### State
: 컴포넌트 안에서 관리되는 데이터

- State 정의
```
    const [data, setData] = useState('초기값')
```

### Props
: 부모 컴포넌트에서 자식 컴포넌트로 전달하는 데이터

- props 전달
```
    <자식컴포넌트 props1={data1} props2={data2}/>
```
- props 전달받기
```
    // 방법 1
    const 자식 컴포넌트 = (props) => {
        const props1 = props.props1
        const props2 = props.props2
        console.log(props1)
        console.log(props2)
    }

    // 방법 2
    const 자식컴포넌트 = ({ props1, props2 }) => {
        console.log(props1)
        console.log(props2)
    }
```
